操作方法
マップの移動　・・・　マウスのドラッグ。矢印ボタン。方向キー（↑↓←→）を使用して
マップを動かして下さい。
目的地の選択　・・・　アイコンで表示されている目的地を右クリックで選択してください。
イベント　　・・・　マップ上にはマウスカーソルが通過するとアイコンが現れる
場所があります。その場所を右クリックすることでイベントが
発生するばあがあります。

SPEED　　・・・　ドラッグ操作によるマップのスクロール速度を設定します。
FLICK　　・・・　ドラッグ操作によるマップの慣性スクロールの有無を設定します。
ARROW　　・・・　矢印ボタンの表示状態を設定します。非表示にするとボタンは押せなくなります。
KEY　　・・・　方向キー（↑↓←→）を押したときのスクロール方向を設定します。
AutoSkip　　・・・　シナリオを進めるための重要な目的地で移動的に移動してくれる便利機能です。

User guide
Map Movement  ...  You can move the map by using the arrow buttons, the cursor keys, 
or by dragging. 
Destination Select  ...  Select your destination by clicking on the icons. 
Events  ...  These may appear at certain locations on the map as you move your cursor 
over them. The event will begin 
when you click on the icon.

SPEED  ...  Configures the drag speed. If you change this option, the map's scroll speed will also change.
FLICK  ...  Configures the inertia for map movement. If you turn this option off, the map will not 'slide' after being dragged.
ARROW  ...  Configures the display of the arrow buttons. If you turn this option off, the buttons will disappear.
Keys  ...  Inverses the scroll direction for the cursor keys（↑↓←→）.
AutoSkip  ...  Automatically moves you to key locations.
